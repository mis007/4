## 📊 项目结构总览

### 📁 完整的文件组织结构

```
src/
├── styles/                    # 🎨 主题系统
│   ├── theme.ts              # 东里村特色主题

│   │   ├── ResponsiveLayout.tsx  # 响应式布局
│   │   └── VillageLayout.tsx     # 主布局
│   ├── pages/               # 页面组件
│   │   ├── VillageHomePage.tsx    # 首页
│   │   ├── VillageLoginPage.tsx   # 登录页
│   │   └── VillageAgentSystem.tsx # Agent系统
│   └── common/              # 通用组件
├── services/                   # 🔧 业务逻辑
│   ├── apiService.ts         # API服务
│   ├── agentSystem.ts       # Agent系统
│   └── ...                 # 其他服务
├── types/                     # 📝 类型定义
├── utils/                      # 🛠️ 工具函数
└── assets/                     # 📁 静态资源
```

### 🎯 核心设计原则

1. **模块化设计**：每个组件职责单一，高内聚低耦合
2. **响应式优先**：移动端体验优先，桌面端功能完整
3. **性能优化**：懒加载、缓存、代码分割
4. **类型安全**：TypeScript严格模式，完整类型覆盖
5. **用户体验**：友好的交互反馈，流畅的动画效果
