## 🚀 快速开始

### 📋 立即启动项目

```bash
# 1. 安装依赖
npm install

# 2. 启动前端开发服务器
npm run dev

# 3. 启动后端API服务器
npm run server

# 4. 访问应用
# 前端：http://localhost:3000
# 后端API：http://localhost:3001/api/health
```
